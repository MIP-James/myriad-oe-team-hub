# MYRIAD Team Hub — 시작하기

이 문서는 처음 프로젝트를 받았을 때 해야 할 일의 **전체 흐름**만 보여줍니다.
각 단계는 아래의 개별 가이드에서 자세히 다룹니다.

## 전체 흐름 (한 번만 하면 됨)

```
1. Node.js 설치              ──→ 컴퓨터에 개발 도구 설치
2. GitHub 계정 + 저장소 생성  ──→ 코드 보관소
3. Supabase 프로젝트 생성     ──→ DB + 로그인 시스템
4. Google OAuth 클라이언트    ──→ 회사 Gmail 로그인 연결
5. 로컬에서 테스트 (npm run dev)
6. Cloudflare Pages 배포     ──→ 인터넷에 공개
```

## 궁금증 FAQ

**Q. 돈 드나요?**
A. 전부 무료 플랜입니다. 팀이 6명이라 충분합니다.
- Cloudflare Pages: 무료 (대역폭 무제한, HTTPS 기본)
- Supabase: 무료 (DB 500MB, 스토리지 1GB)
- GitHub: 무료 (비공개 저장소 포함)

**Q. Supabase가 일시정지된다던데?**
A. 7일간 DB에 요청이 하나도 없으면 자동 멈춤. 팀이 매일 로그인만 해도 자동으로 유지됩니다. 혹시 모를 경우 UptimeRobot 같은 서비스로 5분마다 ping을 보내면 됩니다 (04번 가이드에 방법 포함).

**Q. 유틸리티 EXE는 언제 연결되나요?**
A. 지금은 Phase 1 (웹 기본 뼈대)만 완성. 각 팀원의 PC에서 실행되는 런처는 Phase 4에서 만들 예정입니다.

**Q. 개발자가 아닌데 가능할까요?**
A. 코드는 Claude가 작성. 사용자는 가이드대로 계정 생성 + 키 복사/붙여넣기 + 가끔 버튼 클릭만 하시면 됩니다.
